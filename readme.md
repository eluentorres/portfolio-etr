# Portfolio Personal - Eluen Torres

¡Bienvenido a mi repositorio de Portfolio! Este proyecto es un sitio web responsivo, moderno y limpio diseñado para presentar mi marca personal, mostrar mis habilidades técnicas, organizar recursos útiles y destacar mis desarrollos web de forma interactiva.

##  Características del Proyecto

- **Estructura Modular:** Organizado de forma independiente para facilitar el mantenimiento y escalabilidad de los proyectos.
- **Diseño Totalmente Responsivo:** Adaptado para una visualización óptima tanto en pantallas de escritorio como en dispositivos móviles mediante layouts avanzados (Grid y Flexbox).
- **Interactividad Avanzada:** Implementación de menús dinámicos, selectores de tema (Modo Oscuro/Claro) y utilidades organizadas en pestañas (Tabs).
- **Integraciones con APIs:** Conexión y renderizado de datos en tiempo real (como el widget del clima) para enriquecer la experiencia del usuario.

##  Tecnologías Utilizadas

- **HTML5:** Estructuración semántica y accesible de las páginas web.
- **CSS3:** Estilos personalizados, uso de variables nativas para la gestión de temas visuales y diseño adaptable.
- **JavaScript (ES6+):** Lógica del lado del cliente estructurada de manera limpia y modular (funciones dinámicas para menús, persistencia de tema y consumo de APIs).
