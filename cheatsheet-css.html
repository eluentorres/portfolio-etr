Hoja de Trucos CSS3

Conceptos avanzados y configuraciones clave para dominar el diseño responsivo, arquitecturas modernas y sistemas de color variables.

1. Variables Nativas (Custom Properties)

Permiten almacenar valores reutilizables en todo el documento, facilitando la implementación de características complejas como el modo oscuro:

:root {
  --primary-color: #2563eb;
  --secondary-color: #1e293b;
  --font-base: 'Arial', sans-serif;
}

h1 {
  color: var(--primary-color);
  font-family: var(--font-base);
}

2. Caja Flexible (Flexbox)

Ideal para alinear elementos en una sola dimensión (filas o columnas). Controla la distribución del espacio de forma predictiva:

.container {
  display: flex;
  flex-direction: row; /* u o bien 'column' */
  justify-content: space-between; /* Alineación horizontal */
  align-items: center; /* Alineación vertical */
  flex-wrap: wrap; /* Salto de línea si el espacio falta */
}

3. Rejilla Bidimensional (CSS Grid)

Diseñado para controlar filas y columnas simultáneamente. El uso de auto-fit es excelente para layouts responsivos sin usar media queries:

.grid-layout {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1.5rem;
}
