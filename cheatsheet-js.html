Referencia técnica para manipulación del árbol de elementos, operaciones avanzadas funcionales con colecciones de datos y flujos asíncronos.

1. Selección y Manipulación del DOM

Métodos estándar para interactuar con la interfaz del documento y reaccionar a acciones dinámicas ejecutadas por el usuario:

// Seleccionar elementos
const button = document.querySelector('#theme-toggle');

// Escuchar un evento de clic
button.addEventListener('click', () => {
  document.body.classList.toggle('dark-mode');
});

2. Métodos de Arrays Modernos (Inmutables)

Transformación y filtrado eficaz de colecciones sin alterar la estructura original de los datos:

const skills = [
  { name: 'Git', state: 'dominada' },
  { name: 'Python', state: 'aprendiendo' }
];

// Filtrar elementos
const dominadas = skills.filter(s => s.state === 'dominada');

// Mapear a cadenas de texto HTML
const htmlList = dominadas.map(s => `<li>${s.name}</li>`).join('');

3. Promesas y Flujos de Control Asíncronos

Estructura moderna utilizando async/await para manejar peticiones de red o ejecuciones en segundo plano de manera legible:

async function obtenerDatos(url) {
  try {
    const respuesta = await fetch(url);
    const datos = await respuesta.json();
    console.log(datos);
  } catch (error) {
    console.error("Error en la petición:", error);
  }
}
